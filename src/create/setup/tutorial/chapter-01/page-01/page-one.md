### Add & Subtract
Writing basic functions.

A function has inputs and outputs. The inputs we call "parameters" and wrap them in `(` brackets `)`.

The output is `return`ed from the function.

```
                    // input
function doSomething(parameter) {
  // output
  return parameter;
}
```
